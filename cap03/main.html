Lista de Exercícios - Capítulo 3

Exercício 1

Escreva uma função JavaScript que inverta um número.

Exemplo de entrada: x = 32243

Saída esperada: 34223

Exercício 2

Escreva uma função JavaScript que gere todas as combinações de uma string.

Exemplo de string: 'dog'

Saída esperada: d, do, dog, o, og, g

Exercício 3

Escreva uma função JavaScript que converta uma string qualquer em uma string com letras em ordem alfabética.

Exemplo de string: 'webmaster'

Saída esperada: 'abeemrstw'

Assuma que a pontuação e símbolos não são incluídos na sequência de entrada.

Exercício 4

Escreva uma função JavaScript para clonar um array. Dado um array de entrada, deve ser gerada uma cópia exata como saída e no formato de array.
